Disaster Relief Management System

Programming Language: C++
This project was done for the Computer Algorithms course. This project is designed to streamline disaster relief operations. It helps manage incidents, assign rescue teams, track team movements, and allocate people to shelters. It also handles supplies by allowing users to add and view supply packages. The system uses Dijkstra's algorithm to find the shortest paths for efficient team deployment and shelter allocation.
